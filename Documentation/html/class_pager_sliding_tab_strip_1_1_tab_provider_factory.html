Detailed Description

Default tab provider factory class.

This one supports all the standard functionality: using the ITabProvider implementation from an adapter if it supports it; otherwise falling back to an IconTabProvider provider or TextTabProvider.

If you're looking to customise the behaviour, you can either inherit from this class, or you might want to look at CustomTabProviderFactory, which allows you to supply callbacks for a custom provider, the icon provider and the text provider.
